# Useful information using an iPad

## Shortcuts

### Global
- ⌘ H – Go to Home Screen
- ⌘ space – Search
- ⌘ tab – Switch App
- ⌘ ⌥ D – Show the Dock
- shift tab – Return to the previous field

**While Editing Text:**

- ⌘ A – Select all text
- ⌘ X – Cut
- ⌘ V – Paste
- ⌘ Z – Undo
- ⌘ shift Z – Redo
- ⌥ ⇦ – Move cursor one word to the left
- ⌥ ⇨ – Move cursor one word to the right
- ⌥ shift ⇦ – Select previous word
- ⌥ shift ⇨ – Select next word
